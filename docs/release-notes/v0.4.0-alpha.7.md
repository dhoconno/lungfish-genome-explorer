# Lungfish 0.4.0-alpha.7

This alpha adds graphical Sparkle updates for signed, notarized macOS builds.
After manually installing this release, testers can use **Check for Updates...**
from the application menu for future alpha builds.

The update feed is hosted on GitHub through the fixed `sparkle-alpha` appcast
release, while versioned DMG assets remain attached to each alpha prerelease.
