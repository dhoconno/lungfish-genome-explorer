# Lungfish 0.4.0-alpha.8

This alpha focuses on metagenomics table ergonomics, EsViritu read accounting,
coverage mini-view consistency, and FASTA annotation rendering.

Changes since v0.4.0-alpha.7:

- Made metagenomics result columns fluidly resizable at pixel granularity, with
  hidden zero-width columns recoverable from the column manager.
- Corrected EsViritu unique-hit display so reported detections never show zero
  unique hits when at least one read supports the record.
- Recomputed EsViritu coverage sparklines from each row's own alignment data so
  they match the miniBAM view shape.
- Fixed narrow FASTA/reference bundle annotation rendering so sub-pixel
  annotations remain visible instead of disappearing until the window is wider.
- Fixed local debug app bundle packaging so Sparkle is embedded and the app is
  ad-hoc signed for direct launch.
