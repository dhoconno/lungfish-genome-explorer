Import

  1. Open a bundle.
  2. Choose File > Import VCF to Bundle.
  3. Pick a .vcf or .vcf.gz file.

Explore

  • Use the Variants tab in the bottom drawer.
  • Apply smart filters and query builder rules.
  • Click rows to navigate to variant loci.
