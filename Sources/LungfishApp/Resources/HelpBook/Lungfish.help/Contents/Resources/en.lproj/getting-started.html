Open a Bundle

  1. Choose File > Open.
  2. Select a .lungfish bundle.
  3. Pick a chromosome from the sidebar.

Create a Bundle from NCBI

  1. Open Download Center.
  2. Search for an assembly (for example, rhesus macaque).
  3. Include FASTA and GFF3, and optionally VCF.
  4. Build and open the resulting bundle.

Navigation

  • Type coordinates in the location field.
  • Use zoom controls or gestures to change scale.
  • Open the bottom drawer for tabular inspection.
